# Prusa-Connect-Local
This is a printer connect webservice. It is compatible with another 3D printer services, so many applications which use this service.

### Install
```bash
npm install
```

## OpenAPI specification:

### Run api
  - Documentation (ReDoc)
  - Documentation (SwaggerUI)
  - Swagger Editor
```bash
npm run api:start
```

### Validate api
```bash
npm run api:test
```

## UI

### Clean ui
```bash
npm run ui:clean
```

### Run ui
```bash
npm run ui:start
```

### Build dev
```bash
npm run ui:dev
```

### Build ui
```bash
rm -fr ./dist
npm run ui:build
```
